# cangrejos
Detectar/diferenciar especies de cangrejos mediante capturas de sonido

El programa va cargando todos los archivos en la carpeta que se ingresa en la variable "folderIn".
Genera las subcarpetas "detectados" y "ruido".
En detectados deja los archivos con el recorte de detección.
En ruido deja todo el resto de la señal.

En el estado actual (19/06/2025) se generan tres archivos más que son para verlos en audio y depurar el algoritmo.
